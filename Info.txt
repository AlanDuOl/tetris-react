Component types:
    - Active: tigger changes (direct interaction by users)
    - Reactive: react to changes (reactions can also tigger changes) (indirect interaction by users)

Reactive component
Block:
    actions:
        - restart (ok)
        - move (down) (ok)
        - check for collision:
            - sides: use left + block offsetWidth < display offsetWidth
        - speed up (ok)
        - stop (ok)
        - rotate (ok)
        - move left/right (ok)

    - on render:
        - set local self reference

    - on local self change:
        - set type/layout
        - set tiles/tile size
        - set initial position
        - set initial speed
        - set initial timerSpeed
        - set store block speed to initial value
    
    - on store game gamePaused = true:
        - stop moving:
            - clear current timer
    - on store game gamePaused = false:
        - restart moving with the same speed of stop:
            - reset timer with current timerSpeed

    - on store block speed+:
        - set local speed with current store block speed

    - on local speed+:
        - set timerSpeed with store block speed
    
    - on local timerSpeed change:
        - clear current timer
        - set new timer with new timerSpeed
    

    - on store game level+:
        - set timerSpeed with new level

    
    

Active component
Controls:
    - down:
        - set store block speed+
    - left:
        - set store block turn left
    - right:
        - set store block turn right
    - turn:
        - set store block turn
        

Reactive component
Display-viewport:
    - on mount/render/load:
        - set viewportWidth
        - set viewportHeight

    - on store gameOn = true:
        - restart block
    - on store block restart = true:
        - restart block

Reactive component
Wall:
    - on store game gameOn = false:
        - init
    - store set gameOn = true:
        - nothing
    - store set gamePaused = true | false:
        - nothing
    - on store wall update = true:
        - 

        - update score
        - game over
        - update self
        - restart block

Active component
Menu:
    - Start:
        - store set gameOn = true
        - change self btn textContent = Finish
        
    - Finish:
        - command set gameOn = false
        - change self btn textContent = Start
        - change Resume btn textContent = Pause

    - Pause:
        - store set gamePaused = true
        - change self btn textContent = Resume
    - Resume:
        - store set gamePaused = false
        - change self btn textContent = Pause
